About This Site:

 

This site is builded by Chaochen Wu. The front-end uses Gentelella, an open-source Bootstrap admin template, Echarts, JQuery, and AugularJS. Back-end uses SpringMVC, Spring, and MyBatis. All machine learning algorithm implement by Chaochen Wu and these algorithms are packed into an open source Java library called Vanilla.
