K-Means ClusteringUnsupervised Learning for Clustering

  

STEP 1: Download sample dataset CLICK HERE!

  

STEP 2: Upload File and the data in the file will be read and present in a table.

  ATTENTION:

    1. the first row of data will be automative set as headers.

    2. all data except the first row have to be numeric data

    3. file size limit to 1 MB

    4. Data only allow 9 variables or less.

  

STEP 3: After finish robot test, select how many clusters you want to train and click Start Analysis button. Your data will be training by our platform.

  

STEP 4: Get your training result. You can either view or download your data by different versions.
